Sysem Requirment

First, you need a web server Apache. and make sure rewriting mode is enable

Web Sever

e.g Apache with URL rewriting

Php

Php version 5.6.0 or greater

How to run

Create a virtual host with document root pointing to your project folder and or from your project folder open cmd and run a php developer server from your local on any free port
eg.
Virtual Host http://local.site.com

Php developer server from project folder > run this command php -S localhost:8090 and
access from browser using http://localhost:8090
